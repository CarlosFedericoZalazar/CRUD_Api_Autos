### CRUD API_Cars Frontend

Este es un proyecto de frontend para una aplicación CRUD de gestión de coches. La aplicación permite a los usuarios ver, buscar, crear, actualizar y eliminar coches a través de una interfaz web interactiva.

#### Características Principales
- Visualización de una lista paginada de coches.
- Búsqueda de coches por marca o motor.
- Creación de nuevos coches mediante un formulario.
- Actualización de coches existentes.
- Eliminación de coches con confirmación.
- Uso de SweetAlert2 para alertas y confirmaciones estilizadas.

#### Tecnologías Utilizadas
- HTML5 y CSS3 para la estructura y el estilo de la página.
- JavaScript (ES6) para la lógica del frontend.
- Fetch API para la comunicación con el backend.
- SweetAlert2 para alertas y confirmaciones estilizadas.

#### Estructura del Proyecto
- `index.html`: Archivo principal HTML que contiene la estructura de la página.
- `style/`: Carpeta que contiene los archivos CSS para el estilo de la página.
- `index.js`: Archivo JavaScript que maneja la lógica del frontend y las interacciones con el backend.
- `components/alerts.js`: Módulo JavaScript que contiene funciones para mostrar alertas utilizando SweetAlert2.

#### Instrucciones para Ejecutar el Proyecto
1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web.
3. Asegúrate de que el backend de la API CRUD de coches esté en funcionamiento y accesible desde el frontend.
4. Interactúa con la aplicación para gestionar los coches.

#### Notas Adicionales
- Asegúrate de tener una conexión activa a Internet para cargar la librería SweetAlert2 desde el CDN.
- Personaliza los estilos CSS según tus preferencias para mejorar la apariencia de la aplicación.
